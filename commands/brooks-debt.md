Use the brooks-debt skill to perform a tech debt assessment.
Load the skill via the Skill tool, then classify and prioritize technical
debt using the ten-book decay risk framework and Pain x Spread priority formula
(drawing from The Mythical Man-Month, Code Complete, Refactoring, Clean Architecture,
The Pragmatic Programmer, Domain-Driven Design, A Philosophy of Software Design,
Software Engineering at Google, and Working Effectively with Legacy Code).
Read debt-guide.md from the skill directory for the debt classification
framework and Debt Summary Table template.
